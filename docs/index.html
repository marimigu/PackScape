PackScape

Description

PackScape is a tool for analyzing and packaging scenery for X-Plane 12. It generates HTML reports and creates ZIP archives ready for distribution.

Supports apt.dat version 1200

Features

  1. Converts DSF files to text format (using DSFTool)
  2. Verifies resources and folder structure
  3. Generates HTML reports and backups
  4. Creates ZIP packages ready for release

How to use

Place your scenery folder inside the project directory.

Run the main script:

  • python main.py

Follow the instructions in the terminal.

The packaged ZIP file will appear in the release/ folder. Logs and backups will be saved in backup/.
