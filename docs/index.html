PackScape

Opis

PackScape to narzędzie do analizy i pakowania scenerii dla X-Plane 12. Generuje raporty HTML oraz tworzy gotowe paczki ZIP do dystrybucji.

Wersja apt.dat 1200

Funkcje

  1. Konwersja DSF do formatu tekstowego (DSFTool)
  2. Weryfikacja zasobów i struktury katalogów
  3. Generowanie raportów HTML i backupów
  4. Tworzenie paczek ZIP gotowych do publikacji

Uruchomienie

Umieść katalog scenerii w folderze projektu.

Uruchom główny skrypt:

  • python main.py

Postępuj zgodnie z komunikatami w terminalu.

Spakowana paczka znajdzie się w release/, a logi w backup/.
